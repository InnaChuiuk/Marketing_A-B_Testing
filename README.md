# a-b_testing
# Аналіз A/B тестування маркетингової кампанії

## Опис
Аналіз ефективності рекламної кампанії за допомогою статистичних методів у Python. Ми порівнюємо дві групи користувачів: тих, хто бачив реальну рекламу (**ad**), та тих, хто бачив лише соціальне оголошення (**psa**).

## Мета
Визначити чи є статистично значущою різниця у конверсії між групами.

## Eтапи роботи
**1** Формування збалансованої вибірки (Stratified Sampling) для забезпечення чесного порівняння груп.
**2** Формування гіпотез на обчислення конверсії. Результат показав різницю конверсій 'ad' = 26.4 та 'psa' = 20.2.
**3** Z-test, чи різниця між групами статистично значуща. Результат вийшов менше стандартного p-value 0.05 > 0.039, тобто різниця між групами статистично значуща.
**4** Обчислення агрегатних функцій для кількості переглядів реклами по групам. Середні значення були майже ідентичні (25-24) але медіани менші за значенням (13-12), що вказує на великий розкид даних та є викиди які зміщують середнє значення.
**5** За допомогою візуалізації даних у Boxplot виявлені викиди у обох групах з майже ідентичними переглядами. Тому тестування та заперечення нульової гіпотези про те що 2 групи мають однакову конверсію є абсоютно виправданим, аналіз був чесним і кількість переглядів у обох групах були однакові.
 
## Висновки
* **Реклама працює:** Оскільки p-value 0.039 < 0.05, ми відхиляємо нульову гіпотезу. Приріст конверсії у тестовій групі є статистично значущим, а не випадковим.
* **Чистота тесту:** Аналіз підтвердив відсутність різниці у показах, обидві групи отримували рекламу з однаковою інтенсивністю (медіана 12-13 показів).
* **Оптимізація бюджету:** Наявність значних викидів (користувачі з 100+ показами) вказує на можливість впровадження "частотного ліміту" для економії бюджету без втрати конверсії.
